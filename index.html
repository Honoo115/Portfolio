Quiz of the Second World War

This is my first project. The app is a multiple choice quiz based on historical events that took place during the Second World War. This quiz contains a compilation of Javascript, CSS, and HTML combined together to form a quiz complete with color and text feedback, a flexible design so that it can be viewed on multiple screen sizes, a question + score keeper for progress, and a button to start the quiz over from the beginning.

Find An Anime

This is my second project. The app allows the user to search for an Anime that will result in clickable thumbnails that lead to the 'MyAnimeList' of the respective anime with description, languages, and age included while also saving the seached anime as a search history button should the user wish to re search the previous queried anime by clicking on the button. Also a youtube video showcasing the searched Animes Intro will be viewable on the right hand side allowing the user to watch a clip of the animes Opening sequence to see the art style and animation.
